Algunas funciones requieren import Data.List, Data.Maybe o Data.Ord.

Reduce de der a izq. Ej: foldr (-) 0 [1,2,3] -- 2

Une listas con una función. Ej: zipWith (+) [1] [2] -- [3]

Elimina duplicados (Data.List). Ej: nub [1,1,2] -- [1,2]

¿Todos True? / ¿Alguno True? Ej: or [False, True] -- True

sumNum a => t a -> a

Suma todos. Ej: sum [1,2] -- 3

Máximo / Mínimo de 2 args. Ej: max 1 2 -- 2

Max/Min de una lista. Ej: maximum [1,9,2] -- 9

Igual / Distinto. Ej: 1 /= 2 -- True

Char <-> ASCII (Data.Char). Ej: ord 'A' -- 65

Haskell - Magia Negra de Folds y Rec

Trucos clásicos para finales: Expresar esquemas de recursión en términos de otros.
consta -> b -> a

Ignora el 2do argumento. Ej: const 1 "x" -- 1

foldl via foldr-

Usa foldr para armar una función acumuladora. foldl f z xs = foldr (\x g a -> g (f a x)) id xs z

foldr via foldl-

Idem anterior (crea cadena de comp). foldr f z xs = foldl (\g x a -> g (f x a)) id xs z

foldr via recr-

En recr sobra la sublista, la ignoramos. foldr f z = recr (\x _ rec -> f x rec) z

recr via foldr-

Arrastra la sublista reconstruyéndola en tupla. recr f z xs = snd $ foldr (\x (ys, rec) -> (x:ys, f x ys rec)) ([], z) xs
